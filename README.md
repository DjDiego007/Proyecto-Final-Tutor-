# 🎓 AI English Coach: A1 to C1 Tutor

## 📝 Descripción
Este proyecto consiste en un tutor inteligente basado en Large Language Models (LLMs) diseñado para ayudar a usuarios hispanohablantes a mejorar su nivel de inglés de forma progresiva. El sistema actúa como un mentor que corrige gramática, explica errores y fomenta el uso de vocabulario avanzado.

## 🚀 Funcionalidades
* **Corrección en Tiempo Real**: Identifica y corrige errores gramaticales comunes.
* **Explicación Pedagógica**: Detalla el porqué de las correcciones en un lenguaje sencillo.
* **Escalamiento de Vocabulario**: Sugiere sinónimos de nivel B2/C1 en cada interacción.
* **Memoria de Sesión**: Mantiene el contexto de la conversación para un aprendizaje fluido.

## 🛠️ Tecnologías
* **Python 3.12+**
* **OpenAI API (GPT-4o-mini)**
* **Python-dotenv**: Para la gestión segura de llaves de API.

## 📦 Configuración
1. Clonar el repositorio.
2. Crear un entorno virtual: `python -m venv venv`.
3. Activar el entorno: `.\venv\Scripts\activate`.
4. Instalar dependencias: `pip install -r requirements.txt`.
5. Crear un archivo `.env` y agregar tu `OPENAI_API_KEY`.
